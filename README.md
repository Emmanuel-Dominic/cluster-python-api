# cluster-python-api

This is a basic cluster api flash python project/application

## To run application locally, run:
- [x] create virtualenv & activate it (python3 -m venv venv)
- [x] `pip install requirements.txt`
- [x] `python app.py`

## To run tests
- [x] `pytest -v`
- [x] `coverage run -m pytest`
- [x] `coverage report`
- [x] `coverage html`
